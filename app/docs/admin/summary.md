### 企业上报数据汇总：

### admin/summary/ POST
传入
- name 企业名字

示例：
~~~
{
	"name":"123"
}
~~~

~~~
{
	[
        {
            "start":""  #开始时间
            "end":""  # 结束时间
            "filing":"" #建档期
            "check":"" #调查期
            "diff":"" #差值
        },
        {
            
        }
	]

}
~~~
